###   Chronomètre & Minuteur
Une application React moderne qui combine un chronometre et un minuteur qui permet une bonne gestion de son temps.
<img width="456" height="463" alt="Capture d’écran du 2026-04-18 20-07-08" src="https://github.com/user-attachments/assets/6f7074fa-3dd1-4103-ba3a-3be2c3b6bfd3" />

### Fonctionnalités
- *** Chronomètre** : Compte vers le haut a partir de 00:00
- *** Minuteur** : Compte vers le bas a partir d`une durée choisie par l'utilisateur
- *** Controles** : 
    **Start** : l'utilisateur démarre le chrono ou le minuteur.
    **Stop** : l'utilisateur stop le temps en cours du  chrono ou minuteur.
    **Reset** : l'utilisateur reinitialise le chrono ou le minuteur a 00:00.

### Installation
- Installer Node.js
-Installer pnpm:
 npm install -g pnpm
- Cloner le depot:
   git clone https://github.com/NgassamJoyce/Apprentissage_HTML_CSS_JS/edit/master/chrono-timer
- Lancer l'application:
   pnpm start
- Ouvrir dans le navigateur:
   http://localhost:5173/  

### Utilisation
  *** Mode Chronometre**:
  1. Clique sur le bouton Chronometre
  2. Appuie sur Demarrer (Start) pour mettre le chrono en marche
  3. Utilise Pause (Stop) pour arreter le temps en cours
  4. Reset remet le compteur a 00:00

  *** Mode Minuteur**:
  1. Clique sur le bouton Minuteur
  2. Choisis la Durée (en secondes) dans le champ de saisis
  3. Appuis sur Start pour lancer le compte a rebours
  4. Une Alerte apparait quand il reste 10 secondes
  5.  Un message s`affiche quand le temps est écoulé.
